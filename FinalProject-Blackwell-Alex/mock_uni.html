University Database - It's Live Here!


Premise

The goal of this project was to create a fully functioning database to track students, grades, GPA requirements, and much more. My team and I first began mapping the database in an ERD diagram to visualize connections between entities. We then converted the ERD conversion in terms of SQL. This helped us streamline the relationships into SQL statements. After we finished the SQL statements and inputed some data, we used PHP to fetch our data with SQL queries for the web.
